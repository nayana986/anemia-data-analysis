## About This Project
This is a beginner-friendly data analysis project using Python. The goal of this project is to load, explore, and analyze Complete Blood Count (CBC) datasets (`cbc_anemia_dataset.csv`) to understand health indicators related to Anemia.

## Technologies Used
* **Python**: The main programming language.
* **Pandas**: A powerful Python library used for data manipulation and analysis.

## Current Features
* Loads raw CBC health data from a CSV file.
* Displays the initial rows of the dataset for quick inspection.

## How to Run This Project Locally
1. Clone this repository to your computer.
2. Ensure you have Python installed.
3. Install the required Pandas library by opening your terminal and running:
   `pip install pandas`
4. Run the main script by typing:
   `python main.py`
